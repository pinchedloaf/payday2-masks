payday2-masks
-------------
Payday 2 mod to add mask components.

Requires
--------
IPHLPAPI.dll, v.3.2.0 2015-07-15:1121 version (included).
- Original Author: sirgoodsmoke

Installation
------------
Just copy to pd2 directory. Press F12 for some masks.

This is a one-time use script.
